# RAG Skill

Use this skill when working on retrieval systems.

Principles:

- Retrieval quality beats larger prompts.
- Smaller relevant context beats larger context.
- Avoid duplicate chunks.
- Prefer semantic search.
- Keep chunk size consistent.

Troubleshooting order:

1. Chunking
2. Embeddings
3. Similarity search
4. Prompting
5. Model selection

Metrics:

- Retrieval relevance
- Answer accuracy
- Response latency
